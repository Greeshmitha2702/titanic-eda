# Titanic Dataset - Exploratory Data Analysis

## Project Overview
This project performs Exploratory Data Analysis (EDA) on the Titanic dataset to understand the data, find patterns, and extract insights.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## EDA Steps
- Data loading
- Data cleaning
- Handling missing values
- Data visualization
- Correlation analysis
- Groupby analysis
- Insights

## Key Insights
- Females had higher survival rate than males
- First class passengers had higher survival rate
- Passengers who paid higher fare had higher survival
- Most passengers were between age 20–40
- Younger passengers had slightly higher survival

## Conclusion
EDA helped understand the important factors affecting survival and prepared the data for Machine Learning models.
